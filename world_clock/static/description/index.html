Overview

The basic need for an individual of day to day businessess is to know the time of the countries. This app will be able to show the local date and time of the countries set to odoo backend.

Configuration

  • No additional configuration required

Features

  • World clock an be displayed anywhere on odoo website. It can be found as a building block on edit mode and can be placed any where on website.
  • Clocks can be set on odoo backend.
